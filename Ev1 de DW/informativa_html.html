Fundamentos Teóricos del Lenguaje HTML

¿Qué es HTML?

HTML, que significa HyperText Markup Language (Lenguaje de Marcado de Hipertexto), es el estándar para crear y estructurar páginas web. Se utiliza para describir la estructura del contenido en la web mediante etiquetas y atributos, lo que permite que los navegadores interpreten y muestren el contenido de manera adecuada.

Estructura Básica de un Documento HTML

Un documento HTML típico tiene la siguiente estructura: 


<!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Título de la Página</title>
</head>
<body>
    <h1>Bienvenido a mi Página</h1>
    <p>Este es un párrafo de texto.</p>
</body>
</html>

Componentes Clave

  • Declaración DOCTYPE: Indica al navegador que el documento es HTML5.
  • Elemento <html>: Contiene todo el contenido de la página.
  • Elemento <head>: Contiene metadatos sobre el documento, como el título y la codificación de caracteres.
  • Elemento <body>: Contiene el contenido visible de la página, como texto, imágenes y enlaces.

Etiquetas HTML

HTML utiliza etiquetas para definir diferentes tipos de contenido. Las etiquetas más comunes incluyen:

  • <h1>, <h2>, ... <h6>: Para títulos y subtítulos.
  • <p>: Para bloques de texto.
  • <a>: Para crear hiperenlaces.
  • <img>: Para insertar imágenes.
  • <ul>, <ol>, <li>: Para crear listas no ordenadas y ordenadas.

Ejemplo de Etiquetas


<h1>Título Principal</h1>
<p>Este es un párrafo de texto.</p>
<a href="https://www.ejemplo.com">Visita Ejemplo</a>
<img src="imagen.jpg" alt="Descripción de la imagen">

Atributos

Las etiquetas HTML pueden tener atributos que proporcionan información adicional. Por ejemplo, el atributo href en la etiqueta <a> especifica la URL de destino del enlace. 


<a href="https://www.ejemplo.com" target="_blank">Visita Ejemplo</a>

Atributos Comunes

  • id: Identificador único para el elemento.
  • class: Clase CSS para aplicar estilos.
  • style: Estilos en línea aplicados directamente.

Semántica HTML

HTML5 introduce la semántica, que mejora la estructura del documento al utilizar etiquetas que describen mejor el contenido. Ejemplos incluyen:

  • <header>: Para el encabezado de la página.
  • <footer>: Para el pie de página.
  • <article>: Para contenido independiente.
  • <section>: Para secciones del contenido.

Conclusión

HTML es el bloque de construcción fundamental de la web. Comprender sus fundamentos teóricos es esencial para cualquier desarrollador web. A medida que avances en tus conocimientos, podrás combinar HTML con CSS y JavaScript para crear páginas web más interactivas y estilizadas.
