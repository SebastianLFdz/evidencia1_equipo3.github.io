Fundamentos Teóricos del Lenguaje CSS

¿Qué es CSS?

CSS, que significa Cascading Style Sheets (Hojas de Estilo en Cascada), es un lenguaje utilizado para describir la presentación de un documento HTML. CSS permite controlar el diseño, el color, las fuentes y la disposición de los elementos en la página, separando así la estructura del contenido (HTML) de su presentación visual.

Estructura Básica de una Hoja de Estilo CSS

Una hoja de estilo CSS típica tiene la siguiente estructura: 


h1 {
    color: blue;
    font-size: 24px;
}

p {
    color: black;
    font-family: Arial, sans-serif;
}

Componentes Clave

  • Selección: Define qué elemento(s) se va a estilizar.
  • Propiedades: Especifican los atributos que se van a modificar.
  • Valores: Determinan la nueva apariencia de la propiedad.

Selecciones en CSS

Existen diferentes tipos de selectores en CSS:

  • Selector de tipo: Selecciona elementos por su nombre de etiqueta. Ejemplo: h1, p.
  • Selector de clase: Selecciona elementos con una clase específica. Se define con un punto: .clase.
  • Selector de ID: Selecciona un elemento único con un ID específico. Se define con una almohadilla: #id.

Ejemplo de Selectores


h1 {
    color: red;
}

.clase {
    font-size: 20px;
}

#id {
    background-color: yellow;
}

Propiedades Comunes en CSS

Algunas de las propiedades más utilizadas en CSS son:

  • color: Cambia el color del texto.
  • background-color: Cambia el color de fondo de un elemento.
  • font-size: Cambia el tamaño de la fuente.
  • margin: Define el espacio exterior de un elemento.
  • padding: Define el espacio interior de un elemento.

Ejemplo de Propiedades


p {
    color: blue;
    margin: 10px;
    padding: 5px;
}

Modelo de Caja en CSS

El modelo de caja es un concepto fundamental en CSS que describe cómo se renderizan los elementos. Cada elemento se considera como una caja rectangular que tiene:

  • Contenido: El área donde se muestra el texto o la imagen.
  • Padding: El espacio entre el contenido y el borde de la caja.
  • Borde: La línea que rodea el padding y el contenido.
  • Margin: El espacio exterior entre la caja y otros elementos.

Conclusión

CSS es esencial para el diseño y la presentación de páginas web. Comprender sus fundamentos teóricos es crucial para cualquier desarrollador web. Con CSS, puedes hacer que tus páginas sean visualmente atractivas y mejorar la experiencia del usuario.
